# ReaderRoll 📚

ReaderRoll is a social book-sharing platform designed for readers to share summaries, discover new books, and interact with a community of book enthusiasts.
ReaderRoll is designed as mini university project.


## ✨ Features

* **Book Summaries:** Users can upload and share summaries including book title, author, and page count.
* **Social Feed:** A dynamic homepage where users can view and discover shared summaries from the community.
* **Engagement System:** Interactive "Like" and "Comment" features for every post.
* **User Dashboard:** A personal space to manage uploaded books and track "Recently Read" titles.
* **Category Management:** Organize and filter books based on different genres and statuses.

## 🛠️ Technologies Used

* **Backend:** [PHP](https://www.php.net/)
* **Database:** [MySQL](https://www.mysql.com/)
* **Frontend:** HTML5, CSS3 (Custom Styling), JavaScript
* **Environment:** XAMPP / Apache Local Server

## 💻 Installation and Setup

To run this project in your local environment, follow these steps:

1. **Clone the repository:**
```bash
git clone https://github.com/ibahars/ReaderRoll.git

```


2. **Database Configuration:**
* Open your `phpMyAdmin` panel.
* Create a new database named `readerroll_db`.
* Import the provided `.sql` file located in the project folder.


3. **Project Setup:**
* Move the project folder to your local server directory (e.g., `htdocs` for XAMPP).
* Update the database connection settings in the configuration file (e.g., `db.php` or `config.php`) with your local credentials.


4. **Run the Application:**
* Open your browser and navigate to `http://localhost/ReaderRoll`.
